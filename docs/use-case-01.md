# Извлечение числовых фактов из неструктурированного текста

[Назад к списку компонентов](../README.md)

## Описание

Задача заключается в автоматическом извлечении из неструктурированных текстов заявок на аренду жилья числа проживающих людей. Модель определяет целое значение из текста и возвращает его в виде int.
